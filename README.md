# CatCoin ERC-20 Token

Ptoject that creats and deploys a custom ERC-20 token using Hardhat and OpenZeppelin.

## Features
- Custom ERC-20 token (CatCoin)
- Deployment to Ethereum Sepolia testnet
- Token transfers between wallets

## Tools Used
- Hardhat
- OpenZeppelin
- MetaMask
- Alchemy
- Sepolia Testnet

## Deployment

```bash
npx hardhat run scripts/deploy.js --network sepolia
